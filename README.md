# ecommerce-fraud-detection


## 🧮 SQL Analysis

The dataset was explored using SQL to identify patterns in fraudulent transactions.

### Key steps:
- Data cleaning and IP-to-country mapping
- Exploratory Data Analysis (EDA)
- Transaction characteristics analysis

### Key findings:
- Fraud rate is ~9.5% of all transactions
- Geographic differences exist (some countries higher risk)
- Time-to-purchase is the strongest fraud indicator
- Transaction value and demographics are weak predictors
